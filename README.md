# Practical Bookshelf

Practical Bookshelf is a React and Supabase web application that allows users to manage their personal book collection and reading progress.

Users can:

* Create an account and log in securely
* Add books to their personal library
* Upload cover image URLs for books
* Track reading progress
* Update reading status
* Search books by title or author
* View books by category such as:

  * Currently Reading
  * To Read
  * Completed

---

# Features

## Authentication

* User signup and login powered by Supabase Authentication
* Protected routes prevent unauthorized access to pages
* Logout functionality included

## Book Management

* Add new books
* Delete books
* Update reading progress
* Update reading status
* Store optional cover image URLs
* Reading progress percentage bar

## Dashboard

The Home page displays:

* Total books owned
* Books completed
* Current reads
* To-read books
* Recently added books

## Search

Users can search books by:

* Title
* Author

## Responsive Sidebar

* Collapsible sidebar navigation
* Responsive layout for smaller screen sizes
